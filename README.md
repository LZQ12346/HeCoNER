# HeCoNER

This repository is for the paper **Heterogeneous Collaboration and Instance-Adaptive Constraints for Few-Shot Nested NER**, accepted at **IJCNN 2026**.

## Overview

HeCoNER is a method designed for **few-shot nested named entity recognition**, aiming to better handle complex nested structures under limited annotated data. The framework explores heterogeneous collaboration mechanisms together with instance-adaptive constraints to improve entity recognition performance in low-resource settings.
